# 📦 项目依赖说明

## 初始安装命令

克隆项目后，运行以下命令安装所有依赖：

```bash
npm install
```

## 核心依赖（已包含在 package.json）

项目已配置好，npm install 会自动安装以下依赖：

### 运行时依赖
- **vue** (^3.x) - Vue 3 核心框架
- **vue-router** (^4.x) - Vue Router 路由管理
- **pinia** - Pinia 状态管理

### 开发依赖
- **vite** - Vite 构建工具
- **typescript** - TypeScript 支持
- **vue-tsc** - TypeScript 类型检查
- **@vitejs/plugin-vue** - Vue 插件

## 额外说明

### 为什么需要这些依赖？

1. **vue-router** - 管理页面路由，实现登录验证和页面跳转
2. **pinia** - 轻量级状态管理，管理用户登录状态

### 启动项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build

# 类型检查
npm run type-check
```

### 常见问题

**Q: 运行时报错 "pinia not found"?**
A: 运行 `npm install` 确保所有依赖已安装

**Q: 页面空白?**
A: 检查浏览器控制台是否有错误，确保依赖完整安装

## 技术栈版本

- Vue: 3.4+
- Vue Router: 4.x
- Pinia: 2.x
- Vite: 5.x
- TypeScript: 5.x
